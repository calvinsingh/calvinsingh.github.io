Enhancement Three



  • Enhancement Three: Database

    In the time of code review, I outlined some of the weaknesses resulting from using unsuitable database management skills. The first weakness is that the crime report application database connection mechanism (Mukherjee, 2019). To connect the crime application to the database, it included a different language like PHP. When using different programming language, it makes the connection complex and incompatible with its functions. Another weakness is that the database was hosted locally, which limited the accessibility to various users of the crime application.

    To improve the artifact, I corrected the initial weakness through using the popular Java Database connectivity approach (JDBC), which is easy and utilizes a single programing language; java. When using the JDBC approach, the connection becomes simple with a single file which holds the java connection codes rather than multiple files of different programming language codes. The other improvement involved the hosting of the database into a remote server, which enables multiple access by various users. Also, through remote server hosting technique, data retrieval and data transmission is smooth and takes less execution time unlike local server hosting technique.

    Those two advancements concerning database management skills allowed the full project to meet the application's objectives and requirements. The application now enables universal database accessibility and takes less execution time to load and display crime activity details, unlike the previous application. Through extensive research on database management, I was able to perfect on the field. I worked mostly on the JDBC approach and database management such as remote server technique. It also allowed me to check on efficiency, database management complexity and choosing appropriate database connectivity. Conversely, I still have to work on the JDBC part, to ensure that I get the best JDBC approach depending on the project usage and the targeted computer systems (Tribhuvan, 2020).

  • Achievement

    The improvements outlined in my initial code review have been fulfilled, and the program is now enhanced. Database accessibility is now universal and easy. The application currently allows users to access the database remotely. Conversely, I am still checking on the best remote server to apply so that all users, despite their computers' memory capacity, must have the project functioning and fulfilling the objectives and requirements.
