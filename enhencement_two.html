Enhancement Two



  • Enhancement Two: Algorithms and Data Structure

    In the time of code review, I outlined some of the weaknesses resulting from using unsuitable data structures and algorithms. The first weakness is the luck of proper and ,ore suitable algorithm used when searching for crime reports. The application uses some fewer or almost no algorithms for fetching and organizing data in a proper manner. Due to this way data retrieved is difficult to organize and sort. Further the retrieval of data (huge amount of data) is slow and hence time consuming. Also, the application lacked an appropriate algorithm applied in the project to create search forms for the crime activities' details. Hence, the application depends on the SELECT statement to search for the required information.

    To improve the project, I rectified the initial weakness relating to application slowing down data retrieval when it comes to searching. To enable useful functionality, I developed an Array List to hold the data fetched from database. The data being stored is sorted and organized in a way that only the useful and required information is displayed on the application forms.

    The other weakness was relating to searching the reported crime activities and further details such as the location and type of crime. The searching for the reported crime activities was among the objectives to create the project. Retrieving and displaying of data must be easy and quick. On the previous application, users can search for the data. Conversely, due to the weak algorithms applied, the data fetched, and the execution time during display of the data fails to meet the application objectives and requirements. To rectify that, I initiated a searching algorithm when fetches all necessary data from the database, filtering it and displaying it to the users.

  • Achievement

    Those two improvements concerning data structure and algorithms allowed the full project to meet the application's objectives and requirements. The application now takes less execution time to load and display crime activity details, unlike the previous application. Through extensive research on data structure and algorithms, I was able to perfect on the field (Wei et al., 2020, p. 104619).
    I worked mostly on the java data structures and algorithms such as stacks, queues, list, Binary Trees, Binary search trees, and many more during my research. It also allowed me to check on efficiency, algorithms' time complexity and choosing algorithms on memory use. Conversely, I still have to work on the algorithm part, to ensure that I get the best algorithms depending on the project usage and the targeted computer systems.

  • Update

    I managed to work on all the planned enhancement on this part of the project.
